# 1000번: A+B

## 문제 설명


<p>두 정수 A와 B를 입력받은 다음,&nbsp;A+B를 출력하는 프로그램을 작성하시오.</p>



## 입력 형식


<p>첫째 줄에 A와 B가 주어진다. (0 &lt; A, B &lt; 10)</p>



## 출력 형식


<p>첫째 줄에 A+B를 출력한다.</p>



## 예제

### 예제 입력 1

```
1 2

```

### 예제 출력 1

```
3

```
